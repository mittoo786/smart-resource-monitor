# Smart Resource Monitoring and Prediction System

A Python-based monitoring system that collects real-time CPU and memory usage data, stores it in SQLite, visualizes trends through a Streamlit dashboard, and predicts future CPU utilization using Machine Learning.

## Features

- Real-time CPU monitoring
- Real-time memory monitoring
- SQLite database storage
- Interactive Streamlit dashboard
- Machine Learning CPU prediction

## Technologies Used

- Python
- Psutil
- SQLite
- Pandas
- Streamlit
- Scikit-Learn

## Run

Collector:

python collector.py

Dashboard:

python -m streamlit run dashboard.py
